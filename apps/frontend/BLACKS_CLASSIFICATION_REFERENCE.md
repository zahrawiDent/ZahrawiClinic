## Surface Abbreviations

```
M = Mesial (toward midline)
O = Occlusal (chewing surface, posterior)
I = Incisal (biting edge, anterior)
D = Distal (away from midline)
B = Buccal (toward cheek)
L = Lingual (toward tongue)
```

### Common Multi-Surface Codes

```
MO    = Mesial + Occlusal
DO    = Distal + Occlusal
MOD   = Mesial + Occlusal + Distal
MI    = Mesial + Incisal
DI    = Distal + Incisal
MODBL = All 5 surfaces (full restoration)
```

## Clinical Tips ğŸ’¡

### Class I
- Most common in newly erupted molars
- Check pits and fissures with explorer
- Sealants for prevention
- Conservative prep (minimize removal)

### Class II
- Always check contact with floss
- MOD = "mega filling" (avoid if possible)
- Consider separate MO and DO instead of MOD
- Rubber dam essential

### Class III
- Esthetic concerns (use composite)
- Access from lingual to preserve labial enamel
- Careful with contact point restoration
- Common in children (proximal caries)

### Class IV
- Most challenging esthetic restoration
- Incisal edge involvement complicates
- Consider composite layering technique
- May need beveling for blending

### Class V
- Often non-carious (abrasion/erosion)
- Correct causative factors (brushing, acid)
- No mechanical retention â†’ adhesion critical
- GIC or composite appropriate

### Class VI
- Unusual (wear, not caries)
- Assess occlusion before restoration
- May indicate bruxism
- Often multiple teeth affected

## Historical Note ğŸ“š

**Dr. Greene Vardiman Black (1836-1915)**
- "Father of Modern Dentistry"
- Dean of Northwestern University Dental School
- Published classification in 1908
- Still the standard worldwide today (117+ years!)

Original purpose: Standardize cavity preparation designs for amalgam restorations.

Modern adaptation: Extended to all restorative materials (composite, GIC, etc.) while maintaining classification principles.
